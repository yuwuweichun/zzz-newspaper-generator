# zzz-newspaper-generator（绝区零报纸生成器）
## 如何使用
### 编辑
在上方编辑区域中，根据输入框前的标签内容填写报纸区域中对应的字段内容，下方报纸区域中的字段内容会更新为你填写的字段内容。
### 导出
完成编辑后，点击"导出为图片"即可
